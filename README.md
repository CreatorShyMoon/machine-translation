# Генератор заданий по машинному представлению вещественных чисел

## Описание

Приложение на языке C генерирует индивидуальные задания по теме
машинного представления вещественных чисел (IEEE-754).

Для каждого варианта создаются два файла:

1. Файл с исходными числами
2. Файл с машинным представлением и вычисленной ошибкой

## Входные данные (input.txt)

Файл содержит:

N – количество вариантов  
K – количество чисел в варианте  
Разрядность (32 или 64)  
a – нижняя граница диапазона  
b – верхняя граница диапазона  
P – количество знаков после запятой  

Пример:

20  
5  
32  
-10  
10  
3  

## Выходные данные

Создаются два каталога:

- zadanie/
- proverka/

В каждом каталоге формируются N файлов в формате Markdown.
